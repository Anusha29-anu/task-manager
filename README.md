# Task Manager API

## Features
- User Signup & Login (JWT Authentication)
- Role-based user system
- Create & view tasks
- Protected routes
- MongoDB database
- Deployed on Railway

## API Endpoints
- POST /signup
- POST /login
- POST /tasks
- GET /tasks
